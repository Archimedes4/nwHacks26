# Somnia
## Structure
/frontend Expo project
/backend Server files


Input
Gender: Male Female
Age: integer (1-100)
Sleep Duration: float (of hours)
Physical activity time per day: integer (of minutes)
Height: in cm
Weight: in kg
*Resting Heartrate: integer (50-200)
*Daily steps: integers
*Stress level: integer (1-10, subjective)

Output:
Quality of sleep (1 - 10)

*Starred entries are optional*
